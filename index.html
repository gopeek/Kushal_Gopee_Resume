About me

No bug like the coding bug.

I never really understood my dream. I enjoyed coding growing up but never thought it would take me anywhere in life. Sure I figured a combination of Math and a bit coding could make me a prime time candidate for teaching. Well at least that is what I did when I started out my professional career path.

But I just felt meh. I yearned to learn more, yearned to learn something different. Allow my mind to be creative.

And that's when it happened; the spark to my ignition the instant I clicked "How to build a website with HTML5 and CSS3" I was hooked the love for code came back, Since then the ship has sky rocketed for me. Software engineering calls to all my passions; it incorporates creativity and problem solving and I'm allowed to break it to improve the code (in a separate branch obviously)

I love applying responsive design principles and watching my web pages shrink into mobile screens and still looking amazing. It is oddly satisfying. Additionally creating JSON based API to connect to the front-end. I think in a way I'm kind of like those web pages; I'm moldable , but I still keep my creative flare intact.

My specialties include quickly learning new skills and programming languages, problem solving, responsive design principles, website optimization, and the Model View View Model (MVVM) and Model View Controller (MVC) methods including the famous REST API architecture methods of organizing code.

So far I have JavaScript, HTML5, CSS3, Java, Spring Boot, Spring MVC Python, jQuery, Bootstrap, ReactJs, Grunt, UX/UI design on Figma, Business Analysis, Project Management and Git/GitHub under my belt. I've started learning Node.Js, MongoDB, and Express.JS. I'm still enthusiastically grabbing onto any other programming languages, frameworks, or principles I can integrate into the coding web in my head.
